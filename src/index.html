Range Slider

     
configuration :
1) web component instance
2) Options: {
  min?: number;
  max?: number;
  tooltip?: boolean;
  step?: number;
  value?: number[];
  onChange?:Function
}
3) if attribute option not passed,
defaultOptions = {
  min: 0,
  max: 100,
  tooltip: true,
  step: 0,
  value: [0, 100]
}
will be taken.
4) onChange event support
